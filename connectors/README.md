# Connectors

Client connectors for connecting to PlayHouse servers.

## Available Connectors

| Language | Directory | Target Platform | Status |
|----------|-----------|-----------------|--------|
| C# | [csharp/](csharp/) | .NET, Unity | Active |
| C++ | [cpp/](cpp/) | Unreal, Native | Planned |
| Java | [java/](java/) | Android, JVM | Planned |
| JavaScript | [javascript/](javascript/) | Web, Node.js | Planned |
| Unity | [unity/](unity/) | Unity (UPM) | Planned |
| Unreal | [unreal/](unreal/) | Unreal Engine | Planned |

## Usage

### C# Connector (Unity)

Add via UPM git URL:
```
https://github.com/ulala-x/playhouse.git?path=connectors/csharp
```

### JavaScript Connector

```bash
npm install playhouse-connector
```
